# AI-Ready Data 용어사전

이 위키에서 낯선 단어를 만났다면 여기에 약어나 한글 표현을 입력한다. 정의뿐 아니라
**왜 실무에서 중요한지**와 함께 검색할 표현을 볼 수 있다.

<GlossaryExplorer />

[전체 용어 CSV 내려받기](../downloads/ai-ready-data-glossary.csv) · 우리 조직의 용어를
정리하려면 [업무 용어사전 템플릿](../templates/glossary-template.md)을 사용한다.

::: info 정의의 범위
쉬운 이해를 위한 이 프로젝트의 실무 정의이며 법률·표준의 공식 번역을 대신하지 않는다.
AI·ML 데이터 품질은 [ISO/IEC 5259-1:2024](https://www.iso.org/standard/81088.html),
보안 용어는 [NIST 용어집](https://csrc.nist.gov/glossary), 출처 이력은
[W3C PROV-O](https://www.w3.org/TR/prov-o/)의 원문을 함께 확인한다.
:::
