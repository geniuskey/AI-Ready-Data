새봄기공 AI-Ready 실습 팩
===========================

이 폴더의 회사, 인물, 주소, 설비, 문서, 수치는 모두 교육용 합성 정보입니다.
실제 조직 또는 실제 운영 절차를 나타내지 않습니다.

목표
----
A공장 설비 담당자 8명이 P-101 반복 경보 시 효력 중인 승인 절차와 최근 승인된
유사 조치를 찾아 첫 확인 순서를 결정하도록 돕는 AI 검색의 준비 상태를 만듭니다.

진행 순서
---------
1. 01~05 문제 원천을 읽습니다.
2. 10~12 빈 산출물을 채웁니다.
3. 90~91 정답 예시와 비교합니다.

주의
----
- 실제 사내 문서나 개인정보를 공개 저장소 또는 외부 AI에 업로드하지 마세요.
- 실제 적용 시 승인된 사내 환경에서 최소 범위로 수행하세요.
- 정답 예시는 유일한 답이 아니며, 현업 소유자의 효력 판정을 대신하지 않습니다.
